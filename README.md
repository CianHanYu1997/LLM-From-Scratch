# LLM-From-Scratch

從零開始學習大型語言模型 (LLM) 的核心概念與實作。

## 參考書目

- 《GPT 图解 - 大模型是怎样构建的！》

## 目錄結構

### 01_N-Gram_BoW

文本表示與統計語言模型的基礎，對應書中第一章內容。

| 檔案 | 說明 |
|------|------|
| `bigram.ipynb` | **Bigram 語言模型** - 基於統計的 N-Gram 模型，透過前一個字預測下一個字 |
| `bow.ipynb` | **詞袋模型 (BoW)** - 將文本轉換為詞頻向量，計算文檔相似度 |
| `tfidf.ipynb` | **TF-IDF** - 改進 BoW，降低常見詞權重、提升稀有詞權重 |
